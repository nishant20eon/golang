# Loop

```markdown
for init; condition; post { ... }   // classic
for condition { ... }              // while-like
for { ... }                        // infinite loop
for i, v := range collection { ... } // range loop

// Classic
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// While-like
n := 0
for n < 3 {
    n++
}

// Infinite
for {
    fmt.Println("Forever")
    break
}

// Range
nums := []int{10, 20, 30}
for i, v := range nums {
    fmt.Println(i, v)
}
```
