# Curl

## ✅ Testing with `curl`

After you run the server (`go run main.go`), open a terminal and run this command:

```bash
curl -X POST "http://localhost:8080/v1/details/1" -H "Content-Type: application/json" -d '{"id":1}'
```

### ✅ Output

```json
{
  "id": 1,
  "name": "John Doe",
  "amount": 100.5
}

```
