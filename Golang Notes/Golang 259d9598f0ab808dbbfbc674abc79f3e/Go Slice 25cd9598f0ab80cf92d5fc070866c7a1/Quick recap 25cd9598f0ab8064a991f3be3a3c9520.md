1. **Create a slice**


1. **Access element by index**

```go

```

1. **Find length & capacity**

```go
len(slice_name)   // number of elements
cap(slice_name)   // capacity of slice
```

1. **Add elements**

```go
slice_name = append(slice_name, newValue)
```
